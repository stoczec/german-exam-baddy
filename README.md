## Getting Started

First, run the development server:

```bash
run bun dev
```
install shadcs
```bash
bun x --bun shadcn@latest init
```

install shadcs components
```bash
bun x --bun shadcn@latest add button
```
import Link from:
```bash
import { Link } from "@/i18n/navigation"
```
